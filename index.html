Overview of Software Performance Testing Activities

En général, les tests de performance des applications Web prennent la forme d'envoi de requêtes HTTP. Ces requêtes utilisent généralement les méthodes HTTP les plus courantes, qui sont GET, POST et PATCH, tandis que les autres méthodes comme UPDATE et DELETE sont rarement utilisées.
